2.3 云计算

  1. 简述云计算的三种服务模式。
    1. 基础设施即服务（IaaS）
    2. 平台即服务（PaaS）
    3. 软件即服务（SaaS）
  2. 简述云计算的特点。
    1. 可扩展性
    2. 按需提供资源服务
    3. 虚拟化
    4. 超大规模
    5. 高可靠性和安全性
  3. 简述物联网的体系架构。
    • 感知层、网络层、处理层和应用层
  4. 物联网的关键技术有哪些？
    • 物联网中的关键技术包括识别和感知技术（二维码、RFID、传感器等）、网络与通信技术、数据挖掘与融合技术等。
